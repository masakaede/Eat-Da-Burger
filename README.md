# Eat-Da-Burger

## Description
A restaurant application that lets users input the names of burgers they'd like to eat.

## Installation
1.	Make sure node.js, MySQL Community Server, and MySQL Workbench are installed
2.  Clone this repository
3.  Run the following code at terminal to install package
    > npm install

## Usage
1.  Copy the codes in schema.sql to MySQL workbench to create database table
2.  Make sure you update your mysql user detail and password in the connection.js file
3.	Type the following code at terminal to run the application
	>  npm start
